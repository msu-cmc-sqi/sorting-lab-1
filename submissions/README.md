# Сюда загружайте своё решение

Каждый студент сдаёт работу в отдельной папке:

```

submissions/<your-lastname>/lab1/

```

Пример:
```

submissions/ivanov/lab1/

```

**Важно:** в чужие папки не коммитим.

## Что должно быть внутри

Рекомендуемая структура:
```

submissions/<your-lastname>/lab1/
src/
*.c
*.h
report/
report.pdf
report.tex
README.md   (опционально: как собрать/запустить)

```

Минимальные требования:
- исходники на C в `src/` (`.c/.h`);
- отчёт: `report.pdf` и `report.tex`.
```

### Мини-совет

Если вы делаете сдачу через **fork + PR**, то “кодом” — идеальный вариант.
Если вдруг вы выбрали модель “все пушат в один репо” (не через fork), тогда лучше запретить прямые push в `main` и принимать только PR.

Если хочешь, я так же быстро дам патч-версию для **корневого README**, чтобы он тоже говорил “без zip, сдаём папкой”.
