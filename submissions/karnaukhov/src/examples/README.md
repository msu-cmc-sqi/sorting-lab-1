# examples

В этой папке лежат примеры и шаблоны для отчёта к практикуму.

## Содержимое

- `report_example.tex` — пример/шаблон отчёта в LaTeX.
- `report_example.pdf` — собранный PDF для примера отчёта (как должно выглядеть).
- `Проект задания № 1.pdf` — постановка задачи (то же, что в корне репозитория, если продублировано).
- `Проект задания № 1.docx` — постановка задачи в формате Word (если кому-то удобнее).
- `Трифонов Пильщиков.pdf` — пример готового отчёта (образец оформления).

## Как использовать шаблон отчёта (LaTeX)

1) Скопируйте шаблон к себе в папку сдачи:
```

submissions/<your-lastname>/
report/
report.tex

```

2) Соберите PDF одним из способов:

### Overleaf (проще всего)
- Создайте новый проект на Overleaf
- Загрузите `report.tex` (и картинки/таблицы, если есть)
- Скачайте `report.pdf`

### Локально (если установлен TeX)
```bash
cd submissions/<your-lastname>/report
pdflatex report.tex
pdflatex report.tex
````

> Два прогона нужны, чтобы корректно собрались ссылки/оглавление (если они есть).

## Примечания

* Файлы из `examples/` **не нужно** копировать в Pull Request целиком.
* В сдаче должны быть ваш код (`src/`) и отчёт (`report.pdf` и `report.tex`) — см. README в корне репозитория.
