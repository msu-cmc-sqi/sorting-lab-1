# Сюда загружайте своё решение

Каждый студент сдаёт работу в отдельной папке:

```
submissions/<your-lastname>/lab1/
```

Пример:

```
submissions/ivanov/lab1/
```

**Важно:**

* сдача только через **Fork → Pull Request** (не пушьте напрямую в этот репозиторий);
* в чужие папки не коммитим.

## Что должно быть внутри

Рекомендуемая структура:

```
submissions/<your-lastname>/lab1/
  src/
    *.c
    *.h
  report/
    report.pdf
    report.tex
  README.md   (опционально: как собрать/запустить)
```

Минимальные требования:

* исходники на C в `src/` (`.c/.h`);
* отчёт: `report.pdf` и `report.tex`.
