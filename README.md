# sorting-lab-1 — Практикум №1 «Методы сортировки» (C)

В этом задании нужно **реализовать два метода сортировки массива чисел** и провести **экспериментальное сравнение**
(число сравнений и перемещений) на массивах разной длины и разного типа (упорядочен / обратный / случайный).
Сдача производится через **GitHub Pull Request**.

## Постановка задачи

1. Реализовать **2 метода сортировки** (конкретные методы и вид сортировки — по вашему варианту).
2. Для каждого метода вычислить:

   * **число сравнений** элементов;
   * **число перемещений/обменов** элементов.
3. Генерация исходных массивов реализуется отдельной функцией, создающей массив в зависимости от параметра:

   1. элементы уже упорядочены (1)
   2. элементы упорядочены в обратном порядке (2)
   3. расстановка элементов случайна (3)
   4. расстановка элементов случайна (4) — другой массив
4. Сравнение методов сортировки проводить на **одних и тех же** исходных массивах.
5. Рассмотреть длины:

   * при динамическом выделении памяти: `n = 10, 100, 1000, 10000` (рекомендуется);
   * при фиксированном N: `n = 10, 20, 50, 100`.
6. Результаты оформить на основе нескольких запусков в виде таблицы со средними значениями.

Требования и детали см. в PDF задания.

## Требования к программе (чек-лист)

* [ ] Каждый метод сортировки и генерации массива оформлен отдельной **C-функцией**.
* [ ] Функции не имеют возвращаемых значений (`void`) и принимают два параметра: `n` и массив `a`.
* [ ] Если используется динамическая память, `malloc/free` делаются **до** вызова функций.
* [ ] Программа считает **сравнения** и **перемещения/обмены** для каждого метода.
* [ ] Сравнение проводится на **одних и тех же** исходных массивах.
* [ ] Проведены эксперименты для нужных `n`.
* [ ] В отчёте есть таблица результатов и **средние значения** по нескольким запускам.
* [ ] Код снабжён поясняющими комментариями.
* [ ] В PR лежат: **исходники + report.pdf + report.tex**.

## Подсказка по генерации случайных массивов

Используйте `rand()` и `srand(time(NULL))`.

```c
#include <stdlib.h>
#include <time.h>

// в начале main()
srand((unsigned)time(NULL));
```
