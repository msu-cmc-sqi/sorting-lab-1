# sorting-lab-1 — Практикум №1 «Методы сортировки» (C)

В этом задании нужно **реализовать два метода сортировки массива чисел** и провести **экспериментальное сравнение** (число сравнений и перемещений) на массивах разной длины и разного типа (упорядочен / обратный / случайный).  
Сдача производится через **GitHub Pull Request**.

## Постановка задачи

1. Реализовать **2 метода сортировки** (конкретные методы и вид сортировки — по вашему варианту).
2. Для каждого метода считать:
   - **число сравнений** элементов;
   - **число перемещений/обменов** элементов.
3. Генерировать исходные массивы отдельной функцией, по параметру типа массива:
   1) уже упорядочен  
   2) упорядочен в обратном порядке  
   3) случайный  
   4) случайный (второй прогон с другим массивом)
4. Провести сравнение на одинаковых исходных массивах для обоих методов.
5. Рассмотреть длины:
   - при динамическом выделении памяти: `n = 10, 100, 1000, 10000` (рекомендуется);
   - при фиксированном N: `n = 10, 20, 50, 100`.
6. Оформить результаты нескольких запусков в виде таблицы (средние значения).

Подробные требования и методические указания см. в PDF задания.
